# Elegoo Neptune 4 / 4 Plus – Home Assistant Integration

✅ Fixed **v1.2.1** release.

This version fixes a critical issue where the integration installed correctly but created **no devices or entities**.

## Features
- UI configuration (Config Flow)
- Sensors: status, progress, nozzle temp, bed temp
- Camera entity
- Control buttons (pause / resume / stop)
- Local only (Moonraker / Klipper)
- Home Assistant 2022.5.5+

## Installation
Copy `custom_components/elegoo_neptune4plus` into `/config/custom_components` and restart Home Assistant.
